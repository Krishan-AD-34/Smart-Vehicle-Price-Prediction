# Smart Vehicle Price Prediction 🚗

## 📌 Overview
This project predicts vehicle prices using Machine Learning techniques based on various input features.

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Flask
- HTML, CSS

## ⚙️ Features
- Data preprocessing
- Model training
- Price prediction system
- Web interface using Flask

## 🚀 How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
